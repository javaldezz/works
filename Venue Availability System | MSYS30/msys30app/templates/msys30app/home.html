Welcome to the Venue Availability System

About the Project

This project is developed for the Central Facilities Management Office (CFMO), an office within Ateneo De Manila University that oversees the maintenance, allocation, and scheduling of venues for various events.

This system aims to improve the venue reservation process by enhancing visibility into available venues, reducing inefficiencies, and allowing students to reserve venues more efficiently.

Scope and Features

Scope

This project focuses on improving the CFMO reservation portal by targeting venue availability for LS Venues managed by the CFMO. Testing will focus on Faura Hall.

Features

  • Search for rooms based on date, time, and capacity.
  • Display available rooms in an consolidated format.
